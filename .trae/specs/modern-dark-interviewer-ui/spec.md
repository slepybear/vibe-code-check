# 现代化暗色调面试官面板 UI 重构 Spec

## Why
当前面试官页面（interviewer/page.tsx）UI过于原始和简陋，使用传统的浅色背景（bg-gray-50）和基础卡片样式，缺乏现代感和视觉吸引力。需要重构为现代化的暗色调界面，增强视觉层次感和交互体验。

## What Changes
- **BREAKING**: 完全重新设计面试官页面UI风格
- 采用深色主题（Dark Theme）设计
- 添加玻璃拟态效果（Glassmorphism）
- 增加渐变背景和微妙的动画效果
- 优化卡片布局和层次结构
- 改进交互反馈和hover状态
- 保持原有功能逻辑不变

## Impact
- Affected specs: 无
- Affected code: 
  - `/app/interviewer/page.tsx` - 主要重构目标
  - 可能需要调整全局样式或Tailwind配置

## ADDED Requirements

### Requirement: 现代化暗色调UI
系统 SHALL 提供一个具有以下特征的面试官面板界面：

#### 场景1: 整体布局
- **WHEN** 用户访问面试官页面
- **THEN** 页面应呈现：
  - 深色渐变背景（从深蓝/紫色到黑色）
  - 玻璃拟态效果的半透明卡片
  - 柔和的阴影和边框光晕效果
  - 流畅的过渡动画

#### 场景2: 题目卡片展示
- **WHEN** 浏览题目列表时
- **THEN** 每个题目卡片应具备：
  - 半透明背景配合模糊效果（backdrop-blur）
  - 微妙的边框发光效果（基于难度级别的颜色）
  - hover时的上浮动画和光晕增强
  - 渐变色标签显示难度级别
  - 精致的图标和排版

#### 场景3: 题目详情视图
- **WHEN** 点击查看题目详情时
- **THEN** 详情面板应展示：
  - 更大的玻璃拟态容器
  - 分区块的信息展示（带图标）
  - 表格样式优化（斑马纹、悬停高亮）
  - 复制按钮带动画反馈
  - 平滑的内容切换动画

#### 场景4: 视觉特效与层次
- **WHEN** 与页面交互时
- **THEN** 系统应提供：
  - 卡片hover时的微动画（translateY + shadow变化）
  - 按钮点击的ripple或glow效果
  - 背景添加subtle的动态元素（可选：渐变动画、粒子等）
  - 文字使用渐变或柔和的颜色
  - 合理的间距和padding创造呼吸感

### Requirement: 配色方案
采用以下暗色调配色：

**主色调：**
- 背景：`#0f0f23` (深蓝黑) → `#1a1a2e` (深紫灰) 渐变
- 卡片背景：`rgba(255, 255, 255, 0.05)` + backdrop-blur
- 边框：`rgba(255, 255, 255, 0.1)`
- 主文字：`#e0e0e0` (浅灰)
- 次要文字：`#a0a0b0` (中灰)

**强调色（按难度）：**
- 初级：`#00d4ff` (青色) / `rgba(0, 212, 255, 0.3)` 发光
- 中级：`#a855f7` (紫色) / `rgba(168, 85, 247, 0.3)` 发光
- 高级：`#ff6b6b` (红色) / `rgba(255, 107, 107, 0.3)` 发光

**功能色：**
- 成功/确认：`#10b981` (翡翠绿)
- 警告/提示：`#f59e0b` (琥珀色)
- 信息/链接：`#60a5fa` (天空蓝)

## MODIFIED Requirements

### Requirement: 响应式设计
保持现有的响应式布局能力，但在暗色调下优化移动端体验：
- 移动端卡片堆叠展示
- 触摸友好的按钮尺寸
- 适当的字体大小调整

### Requirement: 性能优化
确保视觉效果不影响性能：
- 使用CSS transform代替位置属性实现动画
- 合理使用will-change属性
- 避免过多的box-shadow层级
